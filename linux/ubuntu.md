Ubuntu
=======

### Customize Ubuntu Terminal

```shell
    export PS1=`\[\033[01;32m\]\u\[\033[01;37m\]:= `
```

+ \h: hostname of the current user
+ \u: username of the current user
+ \v: bash version

Change color:
```shell
    '\[\033[01;32m\]\u'
```

+ 31 Red
+ 33 Yellow
